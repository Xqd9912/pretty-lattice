Glance: quick visualization and analysis

Glance is a local tool that renders crystal structures as beautiful, publication-ready figures — with built-in trajectory, structure, and electronic-property analysis.

Pretty defaults Intuitive GUI Powered by pymatgen Up to 10k atoms
Download Get started GitHub
Features

Everything for the final figure

Focused on viewing and rendering. Prepare structures elsewhere, then bring them here to style, analyze, and export.

🎨

Publication-ready

Tasteful defaults for colors, materials, lighting, and depth.

🖱️

Intuitive GUI

A quiet browser interface for loading, viewing, and exporting structures.

🔬

Reliable parsing

Structure parsing, bonding, and symmetry powered by pymatgen.

🎞️

Trajectory playback

Load XDATCAR, LAMMPS .dump, or .xyz trajectories and step through frames.

📊

Structure analysis

g(r), coordination number, angular distribution, order parameters, MSD, ALTBC.

⚛️

Electronic properties

CHGCAR/ELFCAR isosurfaces, bonding-path profiles, LOBSTER, DOS, and IPR.

Up and running in a minute

Install the Python package, then launch the local GUI.

1

Install from the repository

Requires Python 3.12+. macOS, Linux, or Windows.

2

Launch the GUI

Starts a local server and opens your browser automatically.

3

Open a structure

Drop in a CIF or POSCAR, style it, and export a figure.

Install (pip)
pip install git+https://github.com/Xqd9912/glance.git
Or with uv
uv tool install git+https://github.com/Xqd9912/glance.git
Launch
glance gui
Run once without installing
uvx --from git+https://github.com/Xqd9912/glance.git glance gui
