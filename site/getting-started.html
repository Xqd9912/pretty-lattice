Overview

Glance renders crystal structures as publication-ready figures, built on Three.js.

Beyond visualization, Glance adds trajectory playback, structure analysis, and electronic-property analysis.

Glance focuses on visualization and analysis, not editing. Input files are read-only.

Installation

Glance is a Python package with the frontend bundled in. Install from the repository: 

pip install git+https://github.com/Xqd9912/glance.git

Or with uv as an isolated tool:

uv tool install git+https://github.com/Xqd9912/glance.git

Requirements

  • Python 3.12 or newer
  • macOS, Linux, or Windows
  • Any modern browser
The analysis modules are not on PyPI — install from this repository.

Quick start

After installation, start the local GUI:

glance gui

Glance starts a local server and opens your browser automatically.

Run once without installing

uvx --from git+https://github.com/Xqd9912/glance.git glance gui

Useful launch options

glance gui --no-open     # start the server without opening a browser
glance gui -p 0          # choose any available port automatically

Material & color presets

Tune materials, colors, radii, opacity, lighting, and orientation with presets.

Trajectory visualization

Load XDATCAR, LAMMPS .dump, or .xyz trajectories and step through frames with the built-in player.

Structure analysis

Compute structural and dynamical descriptors over a frame range as interactive charts:

  • Pair distribution g(r)
  • Coordination number
  • Angular distribution
  • Order parameters
  • Mean squared displacement
  • ALTBC

Bond cutoffs are seeded from the first minimum of each g(r) and are editable.

Electronic properties

Analyze and visualize electronic-structure output from VASP:

  • Charge density (CHGCAR) — a 3D isosurface with slices and the LED distribution.
  • Electron localization (ELFCAR) — the same pipeline applied to the ELF.
  • Bonding-path profile — average ELF or charge along a bond between an atom pair.
  • LOBSTER bonding analysis — BWDF and integrated COHP/COOP against bond length.
  • Density of states (TDOS.dat) — the total DOS as an energy–DOS line chart.
  • Inverse participation ratio (vasprun.xml) — the IPR of every state on a shared energy axis with the DOS.
